compatibility.rb
¶

recap isn’t intended to be compatible with tasks (such as those within the bundler or whenever projects) that are built on the original capistrano deployment recipes. At times though there are tasks that would work, but for some missing (and redundant) settings.

Including this recipe adds these legacy settings, but provides no guarantee that original tasks will work. Many are based on assumptions about the deployment layout that no longer hold true.

  extend Recap::Support::Namespace
